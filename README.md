# Setup & Run
- Clone the code of `assignment-app` from  here :point_right: https://github.com/deep-2211/assignment-app.git
```
 git clone https://github.com/deep-2211/assignment-app.git
```
- Open the terminal go the project folder and run :point_down:
```
npm install
```
- On installation done run :point_down:
```
npm run start
```
